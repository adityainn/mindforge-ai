## 📖 Overview

MindForge AI is an intelligent study planner designed to help students stay productive and improve learning efficiency.

The application combines task management, Pomodoro sessions, flashcards, AI-powered quizzes, and personalized study schedules into one modern dashboard.

---

## ✨ Features

### 📋 Smart Task Manager
- Create, edit and delete tasks
- Priority management
- Due dates
- Categories
- Pomodoro estimation

### 🍅 Pomodoro Timer
- Focus timer
- Break timer
- Session tracking
- Productivity statistics

### 🧠 Flashcards
- Create decks
- Card flipping
- Spaced Repetition (SRS)
- Review history

### 🤖 AI Quiz Generator
Generate multiple-choice quizzes from your notes using Gemini AI.

### 📅 AI Daily Planner
Generate a personalized study schedule based on:
- Available study time
- Exams
- Assignments
- Priorities

### 📊 Analytics
- Study time
- Weekly progress
- Completed tasks
- Productivity charts
- Study streaks

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/adityainn/mindforge-ai.git
```

Move into the project

```bash
cd mindforge-ai
```

Install dependencies

```bash
npm install
```

Create a local environment file

```bash
cp .env.example .env.local
```

Add your Gemini API key

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
